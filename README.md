# Ticketing System Lab (Jira)

## Overview
This repository documents a **ticketing system lab environment** built using **Jira** in a **VirtualBox** setup. The lab simulates a real-world IT service desk using **Windows Server 2019**, **Active Directory**, and **Windows 10** clients.

---

## Environment
- **Host OS:** Windows 10  
- **Virtualization:** Oracle VirtualBox  
- **Server OS:** Windows Server 2019  
- **Client OS:** Windows 10  
- **Ticketing System:** Jira  
- **Identity Management:** Active Directory  

---

## Key Objectives
- Deploy and configure Jira in a virtualized environment
- Implement Active Directory for centralized user management
- Simulate IT support ticket workflows
- Document system configuration and setup

---

## What This Lab Demonstrates
- Windows Server administration
- Active Directory setup and domain management
- Ticketing system deployment
- Virtual machine and network configuration
- Technical documentation using GitHub
